REST API – Fall 2021

Designed a REST API that would access a MySQL database containing information about servers running for clients, then display JSON with that information. Website contained cgi script such that when a POST request was recieved, a new AWS ec2 instance would be instantiated for client, which would then run bash script to install everything necessary to start a Minecraft server. Server would then be added to the database and be displayed in the API. Scripting for API was done in python and javascript.

What You See is What You Get Text Editor – Fall 2022

Produced a GUI based text editor with partner doing pair programming. Editor allowed for image, shape and character insertion and deletion by using composite design pattern and storing all data as glyphs. Text editor contained a spell checker, such that when a word was entered, if not in the dictionary, it would highlight red underneath to display to user it was mispelled. Implemented in Java using Swing and AWT libraries.

MIPS Single-Cycle CPU Simulation – Fall 2022

Coded a 32-bit single-cycle CPU simulation using MIPS instruction set. Allows most basic instructions (e.g beq, add, etc...) to be executed. Implemented in C using functional programming.
